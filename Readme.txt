=>this game is for 10+ students it alters questions between add,subtract,multiply with
  operands between 0 to 99.
=>Move pikacho to right and left using the arrows.
=>Stand under the right number to collect it.
=>Remember:the numbers falling are from zero to nine you collect 
  the answer from Left to Right.
=>When you collect the answer press enter to grade it(right or wrong).
=>If you collect four digits the number will be graded automatically.(no ans. is greater than that).
=>You lose if you get 3 wrong answers and you win after 9 right answers.
=>To return to main menu and restart press Esc.
=>Be carfull where you stand the number you take cannot be removed from the answer.
=>run the batch called 'run' to run the main game and graphics file.
